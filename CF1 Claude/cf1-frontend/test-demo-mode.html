CF1 Demo Mode Testing

Instructions:

  1. Open the CF1 platform at http://localhost:3000
  2. Open browser developer console (F12)
  3. Use the buttons below or run commands in the console
  4. Check the Launchpad and Governance pages for mock data

Demo Mode Controls:

Commands to run in CF1 platform console:

// Check current demo mode status
CF1Debug.checkDemoMode()

// Enable demo mode with mock data
CF1Debug.enableDemoMode('investor_presentation')

// Disable demo mode (clear mock data)
CF1Debug.resetDemoMode()

// Test data refresh
// 1. Check /launchpad page (should be empty in dev mode)
// 2. Check /governance page (should be empty in dev mode)
// 3. Enable demo mode
// 4. Check pages again (should have mock data)

Expected Behavior:

  • Development Mode (default): No mock proposals in Launchpad or Governance
  • Demo Mode Enabled: Mock proposals appear in both sections
  • Demo Mode Disabled: Mock proposals disappear immediately
